Sequence Detector (1110)

<img width="782" height="484" alt="image" src="https://github.com/user-attachments/assets/9c520660-ffb9-46cb-8ef7-7206b9a380b5" />

Designed and implemented a Sequence Detector using Verilog HDL to detect the binary sequence 1110 using a Finite State Machine (FSM) approach. The design uses present state and next state logic to track the incoming serial input sequence and generate an output signal when the required pattern is identified.

The implementation supports sequential operation using clock and reset inputs, where state transitions occur synchronously with the clock signal. The detector continuously monitors serial input data and raises the detected output whenever the sequence 1110 appears. The functionality of the design was verified through simulation using a dedicated Verilog testbench and waveform analysis.
